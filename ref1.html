Reflection 1

Discuss your learning objectives for this semester: how do you plan to achieve them? You can think about why you chose this course. You can use the learning outcomes listed in the syllabus for your reference.

My learning objective this semester is to be able to use the Chinese language for daily conversation in order to prepare myself for the future in the case where I am interested to continue my further studies in China. I want to be able to use the Chinese language to describe things and deliver ideas using the correct grammatical structure. I also want to know more about the Chinese language on its own and the culture too. Last but not least, I want to use this course as a way for me to have a supervised practice area so that I can improve my skills faster.

I can achieve all these by participating more in class, ask questions whenever necessary, and try to use Mandarin in-class when talking to classmates and also to Chen 老师。
